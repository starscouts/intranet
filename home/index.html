Welcome to the Equestria.dev Intranet

The new Equestria.dev Intranet, running on 10.0.0.0/8 (as reserved by IANA), is accessible from Equestria.dev's servers, Wi-Fi or a VPN. The server you are using right now is known as 10.0.2.1 (Wi-Fi VLAN) or 10.0.1.1 (servers VLAN). Please see the infrastructure table below for details:

10.0.2.0/24 3 The network Wi-Fi devices connect to. While it behaves similarly to other networks, it has higher security in the event of a compromised device. Yes
